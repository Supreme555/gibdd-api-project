# Инструкции по запуску проекта

## Backend

# Для установки зависимостей используйте команду:

npm install

# Для запуска backend проекта выполните команду:

node index.js


## Frontend

# Для установки зависимостей используйте команду:

yarn 

# Для запуска frontend проекта выполните команду:

yarn dev
